# ErCe-s-Elmas-Toplama-Oyunu

Unity ve C# kullanarak geliştirdiğim 2 seviyeli 2D platform oyunudur. Oyuncu 1. seviyede süreye karşı yarışıyor, 2. seviyede ise zorlu engelleri aşarak veya portal kullanarak oyunu tamamlamaya çalışıyor. Oyuncu hareketleri, çarpışmalar ve basit UI mekanikleri uygulanmıştır. Oyun geliştirme ve nesne yönelimli programlama (OOP) deneyimi kazandım.

## Özellikler
- 2 seviye tasarımı:
  - **Level 1:** Süreye karşı yarış
  - **Level 2:** Engeller ve portal sistemi
- Can sistemi (3 can)
- Oyuncu hareket ve çarpışma mekanikleri
- Menü ve UI yönetimi
- Oyun akışı ve level geçişleri

## Kullanılan Teknolojiler
- Unity 2D
- C#
- Nesne Yönelimli Programlama (OOP)
